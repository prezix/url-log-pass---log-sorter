Как использовать:
    - установить Python последней версии (поставить галочку Add to PATH при установке)
    - извлечь архив
    - установить необходимые зависимости из файла requarements.txt
      открыть консоль, прописать путь к папке со скриптом и ввести команду pip install -r requarements.txt
    - в папку data поместить логи, логи должны быть разархивированны (каждый в своей папке)
    - в консоли прописать путь к папке со скриптом и запустить его (python main.py)
